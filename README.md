# 📱 InsightLog

Aplicativo mobile desenvolvido em React Native com o objetivo de registrar atividades e apresentar indicadores simples a partir dos dados inseridos pelo usuário.

---

## 🚀 Sobre o projeto

O InsightLog é um aplicativo de monitoramento de atividades que permite ao usuário registrar eventos ao longo do tempo e visualizar métricas básicas, como volume de registros e distribuição por categoria.

A proposta do projeto é unir conceitos de desenvolvimento mobile com uma abordagem orientada a dados, oferecendo uma visão simplificada e estruturada das informações cadastradas.

---

## 🎯 Objetivo

Permitir que usuários registrem atividades e acompanhem seus dados por meio de indicadores simples, facilitando a identificação de padrões e o entendimento das informações ao longo do tempo.

---

## 🧠 Funcionalidades (planejadas)

- Registro de atividades
- Listagem de eventos cadastrados
- Categorização dos registros
- Visualização de indicadores (dashboard simplificado)
- Integração com API (etapas futuras)

---

## 🛠️ Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- React Navigation *(planejado)*
- Axios *(planejado)*
- AsyncStorage *(planejado)*
